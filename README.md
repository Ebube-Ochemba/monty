## Environment

- Language: C
- OS: Ubuntu 20.04 LTS
- Compiler: gcc 4.8.4
- Style guidelines: [Betty style](https://github.com/holbertonschool/Betty/wiki)

## How to use

```sh
$ git clone https://github.com/Ebube-Ochemba/monty
$ cd monty
$ gcc -Wall -Werror -Wextra -pedantic \*.c \*.h -o monty
$ ./monty <./path/to/test_file>
```
